# ⚡ Smart Energy Monitoring and Cost Estimation using ESP32

An ESP32-based IoT project that monitors real-time electrical parameters such as voltage, current, power, and energy consumption, while estimating electricity cost based on a predefined electricity tariff.

---

## 📌 Project Overview

This project helps users monitor electrical energy usage in real time using an ESP32 microcontroller, current and voltage sensors. The measured data is processed to calculate power consumption, energy usage (kWh), and estimated electricity cost. The information is displayed locally on a 16×2 LCD and can be integrated with Blynk IoT for remote monitoring.

---

## ✨ Features

- Real-time voltage monitoring
- Real-time current measurement
- Live power calculation
- Energy consumption (kWh)
- Electricity cost estimation
- LCD display output
- ESP32-based embedded system
- IoT-ready architecture

---

## 🛠 Hardware Used

- ESP32 Development Board
- ACS712 Current Sensor (5A)
- ZMPT101B Voltage Sensor
- 16×2 LCD with I2C Module
- Breadboard
- Jumper Wires
- AC Load (Bulb)
